Projects I'm proud of

The First Project

Youtube

In my first project, I developed a YouTube clone using HTML and CSS. Initially, I added pseudo-classes everywhere and had a lot of irrelevant code. Recently, I reviewed and cleaned up the code, removing unnecessary parts and minimizing it. I learned key web development concepts such as `div`, `grid`, `flex`, tooltips, and buttons. Flexbox created a responsive header and sidebar, while CSS Grid ensured a uniform video layout. I styled thumbnails, titles, usernames, and stats, adding tooltips, hover effects, and custom buttons for interactivity. The design is responsive, adapting to various screen sizes. Due to this project, I gained new knowledge on the smallest units of web development.

The Second Project

Instagram

In my second project, I cloned my Instagram page after completing my YouTube clone project. I did everything on my own, applying the skills I learned previously. This project involved using HTML and CSS to create the layout and design, including the header, sidebar, profile section, and photo grid. I implemented Flexbox for the responsive header and sidebar, and CSS Grid for arranging the posts and story highlights. I also added a small JavaScript function to zoom in on images when clicked. This project further solidified my understanding of web development fundamentals and allowed me to explore basic JavaScript interactivity.
